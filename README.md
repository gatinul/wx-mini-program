# wx-mini-program
一个小程序demo

## 组件使用

- view
  - 大部分.wxml文件都用view包裹
- swiper
  - 城市页放了一个swiper（swiper放在view里会无效）
- text
  - 放置文本内容
- picker
  - 选择组件，这里用来选择地区
- image
  - image中mode 有 13 种模式，其中 4 种是缩放模式，9 种是裁剪模式。

## api应用

- navigateTo
  - 保留当前页跳转新页面，url可接参数，用?隔开
